# JOKI GARONG - Retro Pixel Website

![Website Preview]()

## 🎮 About

A retro pixel-style website for JOKI GARONG services. Built with modern web technologies but styled with a nostalgic 90s aesthetic.

## 🚀 Features

- 🎨 Retro pixel design with CRT effects
- 📱 Fully responsive (mobile & desktop)
- 🌍 Multi-language support (Indonesia, English, Japanese, Portuguese)
- ⚡ Built with Vite + Vue 3
- 🖼️ Optimized images with lazy loading
- 🎭 Glitch text animations & scanlines
- 📦 Modular component architecture

## 🛠️ Technologies

- **Vue 3** - Progressive JavaScript framework
- **Vite** - Next generation frontend tooling
- **Tailwind CSS** - Utility-first CSS framework
- **Google Fonts** - Press Start 2P, Rubik Mono One

## 📦 Installation

```bash
# Clone repository
git clone https://github.com/YOUR_USERNAME/joki-garong.git

# Enter project directory
cd joki-garong

# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
